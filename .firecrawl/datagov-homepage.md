# מאגר הנתונים הממשלתי

אתר מאגרי המידע הממשלתיים מרכז מאגרי נתונים מכלל משרדי הממשלה הפתוחים לשימוש הציבור, מאפשר חיפוש נתונים בקלות וביעילות.

## דוגמאות לצריכת API

- ### שאילתה עם דיפדוף



ביצוע שאילתה עם דפדוף מאפשרת לשלוט על כמות הנתונים שחוזרים ועל מספר השורה ממנו רוצים להתחיל את צריכת הנתונים

ה – offset מסמן את השורה שבה מתחילים
ה – limit מסמן כמה שורות יוצגו

הצג שאילתה

- ### שאילתה לפי שדות



פניית Api עם הערך filter המאפשרת לסנן את המידע לפי הערכים המבוקשים

הצג שאילתה

- ### צריכת נתונים על ידי crawling



ניתן להוסיף את האפשרות ל \- user agent קיים או להגדיר user agent חדש.

הצג שאילתה

- ### שאילתה סינון מספר ערכים לפי עמוד



שאילתה סינון מספר ערכים לפי עמודה

הצג שאילתה

- ### שאילתה המחזירה שדות ספציפיים עם פילטור



הגדרת שאילתה שתבצע פילטור על המאגר ותחזיר שדות ספציפיים


הצג שאילתה

- ### שאילתה המחזירה תאריך עדכון אחרון



שאילתה המחזיר תאריך עדכון אחרון

שם יש ערך בשם: last\_modified

הצג שאילתה

- ### רשימת הארגונים



שאילתה לקבלת רשימה של כל הארגונים


הצג שאילתה

- ### הצגת שם ארגון



הצגה של ארגון ע"י שם הארגון


הצג שאילתה

## מאגרים פופולריים

- [**מילוני מאסטר (Code Lists)**](https://data.gov.il/datasets/lamas/dictionaries)



רשימות של ערכים וסיווגים סטנדרטיים ואחידים (כל ערך ברשימה הוא חד-ערכי).
תהליך קביעות מילוני מאסטר מתבצע ע"י הלשכה המרכזית לסטטיסטיקה תוך רצון לבנות שפה אחידה בין משרדי ממשלה.
תהליך זה כולל לימוד תוכן המילון והתייעצות עם גורמים מומחים בנושא, עם ההמועצה הציבורית לסטטיסטיקה ובדיקת סיווגים בינלאומיים, הצעה למילון מאסטר, אישור המילון במועצת המשילות בלמ"ס, פרסום המילון והטמעתו.




![organization image](https://data.gov.il/_next/image?url=https%3A%2F%2Fdata.gov.il%2Fuploads%2Fgroup%2F2021-08-17-143542.259750--.png&w=96&q=75)

- [**כלי רכב עם תג חניה לנכה**](https://data.gov.il/datasets/ministry_of_transport/rechev-tag-nachim)



מאגר זה מכיל את מספרי הרכבים המורשים לשאת תג חניה לנכה של משרד התחבורה.



![organization image](https://data.gov.il/_next/image?url=https%3A%2F%2Fdata.gov.il%2Fuploads%2Fgroup%2F2021-08-18-112030.121138MOTlogo1.png&w=96&q=75)

- [**מספרי רישוי של כלי רכב פרטיים ומסחריים**](https://data.gov.il/datasets/ministry_of_transport/private-and-commercial-vehicles)



מאגר זה מכיל מידע רשמי של מספרי רישוי של כלי הרכב הפעילים הפרטיים משנת יצור 1996 ומעלה ומספרי הרישוי של כלי הרכב הפעילים המסחריים במשקל עד 3,500 ק"ג משנת יצור 1998 ומעלה.



![organization image](https://data.gov.il/_next/image?url=https%3A%2F%2Fdata.gov.il%2Fuploads%2Fgroup%2F2021-08-18-112030.121138MOTlogo1.png&w=96&q=75)

- [**מאגר חברות \- רשם החברות**](https://data.gov.il/datasets/ministry_of_justice/ica_companies)



הקובץ מכיל את רשימת החברות הרשומות במרשם שמנהל רשם החברות כולל חברות בסטטוס: פעילה, פעילה/בפירוק \- בכינוס נכסים, פעילה/בפירוק \- הליך פשרה או הסדר/פירוק זמני, בפרוק מרצון, בפרוק ע"י בימ"ש, מחוקה, מחוסלת מרצון, חיסול ע"י בימ"ש, מחוסלת עקב מיזוג, פעילה זמנית, נגרעה מהמרשם.
הקובץ כולל את מספר התאגיד, שם התאגיד, שם התאגיד באנגלית, סוג התאגיד, סטטוס התאגיד, תת סטטוס התאגיד, תיאור התאגיד, מטרת התאגיד, תאריך התאגדות, מגבלות, מפרה, דוח שנתי וכתובת.

לפרטים אודות רישום והסרה של הערת חברה מפרה: https://www.gov.il/he/service/law\_violationg\_company
